# Roadmap do MVP

## Fase 0 - Base do projeto

- Monorepo com app web, pacote compartilhado, pasta Supabase e documentacao.
- Configuracao inicial de Next.js, TypeScript, Tailwind, ESLint e pnpm.
- Regras de seguranca e escopo inicial documentados.

## Fase 1 - Supabase

- Modelagem inicial do banco.
- Autenticacao.
- Politicas RLS.
- Seeds minimos para desenvolvimento.

## Fase 2 - Autenticacao e layout

- Login.
- Rotas protegidas.
- Estrutura visual base do painel.

## Fase 3 - Clientes e quiosque

- Cadastro e consulta de clientes.
- Tela simples para tablet/quiosque com nome e telefone.
- Validacoes basicas.

## Fase 4 - Manutencoes

- Cadastro de aparelho e problema relatado.
- Status da manutencao.
- Historico de atendimento.

## Fase 5 - Alertas e mensagens

- Alertas de entrega e manutencao.
- Modelos de mensagens para WhatsApp manual.
- Copiar mensagem pronta para envio manual.

## Fora do MVP inicial

- Estoque.
- Vendas.
- Promocoes em massa.
- API oficial do WhatsApp.
- Financeiro.
- Multi-loja avancado visual.
